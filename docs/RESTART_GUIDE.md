# 🔄 Перезапуск 3D MONOLITH после изменений

## ⚠️ Важно!

После внесения изменений в файлы:
- `main.js`
- `preload.js`
- Любые модули в `src/modules/`

**Необходимо перезапустить Electron!**

---

## 🚀 Как перезапустить

### Вариант 1: Через терминал

```bash
cd "C:\Users\murad\Desktop\3D MONOLITH"
npm start
```

### Вариант 2: Если приложение уже запущено

1. **Закройте приложение** полностью
2. **Проверьте**, что процесс завершён:
   - Откройте Диспетчер задач (Ctrl+Shift+Esc)
   - Найдите "3D MONOLITH" или "electron"
   - Завершите процесс, если он есть

3. **Запустите заново**:
   ```bash
   npm start
   ```

---

## ✅ Проверка изменений

После перезапуска откройте консоль (F12) и проверьте:

### 1. Проверка execCommand

```javascript
// В консоли
window.electron.execCommand('echo test')
// Должно вернуть: { success: true, stdout: 'test\n' }
```

### 2. Проверка UVTools

```javascript
// В консоли
window.UVToolsBridge.init()
// Если UVTools установлен: { available: true, ... }
// Если нет: { available: false, ... }
```

### 3. Проверка SLACostCalculator

```javascript
// Загрузите PWMO файл через интерфейс
// В консоли должны появиться логи:
// 🔍 [SLA] Сигнатуры файла:
//    magic32: 0x...
//    magic16: 0x...
//    str4: "..."
```

## 🐛 Если что-то не работает

### «exec-command не найден»

**Решение:**
1. Убедитесь, что `main.js` сохранён
2. Полностью закройте Electron
3. Проверьте, что процесс завершён (Диспетчер задач)
4. Запустите `npm start` заново

### «UVToolsBridge не определён»

**Решение:**
1. Проверьте, что файл подключён в `index.html`:
   ```html
   <script src="src/modules/uvtools-bridge.js"></script>
   ```
2. Обновите страницу (F5)
3. Проверьте консоль на наличие ошибок

### «Не удалось распарсить PWMO»

**Решение:**
1. Откройте консоль (F12)
2. Загрузите PWMO файл
3. Скопируйте логи:
   ```
   🔍 [SLA] Сигнатуры файла:
      magic32: 0x...
      magic16: 0x...
   ```
4. Отправьте логи разработчику для анализа

## 🔧 Отладка

### Включить подробные логи

В `index.html` найдите функцию `loadPWMOWithCalculator` и раскомментируйте:

console.log('🔍 [PWMO] Полный дамп данных:', pwmoData);

```javascript
// В консоли после загрузки файла
const pwmo = window.currentPWMO;
console.log('PWMO данные:', pwmo);
console.log('Метод:', pwmo.method);
console.log('Формат:', pwmo.format);
console.log('Время:', pwmo.printTime, 'ч');
console.log('Объём:', pwmo.volume, 'см³');
```
