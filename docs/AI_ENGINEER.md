## 🎯 ОБЗОР

**3D MONOLITH AI Engineer** — это локальный ИИ-помощник, специализирующийся на аддитивных технологиях (FDM/SLA 3D-печать).

### Миссия
Обеспечение безопасности процессов, помощь в настройке оборудования, расчете параметров и устранении неисправностей.

### Принцип работы
> «Безопасность и точность важнее креативности»

ИИ не является разговорным собеседником на общие темы — это инструмент технической поддержки производства.

## 🛡️ СИСТЕМА БЕЗОПАСНОСТИ

### Уровень 1: Системная инструкция
В промпт жестко зашиты правила:
- Никогда не советуй отключать питание при нагретых компонентах
- Всегда предупреждай о токсичности материалов
- Всегда упоминай СИЗ при работе с химикатами

### Уровень 2: Красные линии (Hard Rules)
Список запрещенных действий, проверяемый программно:

```javascript
// Пример паттерна опасного запроса
/redLines = [
    {
        pattern: /отключ[ия]и?т?[ью]?\s+(питани[ея]|принтер)/i,
        warning: 'Отключение питания во время печати может привести к поломке!'
    },
    {
        pattern: /разбира[йт]?[ия]и?т?[ью]?\s+(хотэнд|сопло)/i,
        requires: ['осты', '30°C', 'холод'],
        warning: 'Разборка возможна только после остывания (<30°C)!'
    }
]
```

### Уровень 3: Валидация ответа
Промежуточный программный фильтр сканирует сгенерированный текст на наличие опасных паттернов перед показом пользователю.

### Уровень 4: Пользовательский контроль
Кнопка «Проверить безопасность» для принудительной валидации последнего совета.

## 📚 БАЗА ЗНАНИЙ (RAG)

ИИ не полагается на внутренние веса для технических данных. Все факты извлекаются из структурированной базы знаний.

### Источники данных

| Источник | Данные |
|----------|--------|
| **Парк оборудования** | Характеристики принтеров, область печати, кинематика |
| **Склад материалов** | Остатки, цены, свойства (PLA, ABS, PETG, фотополимеры) |
| **Текущий заказ** | Параметры расчета (слой, заполнение, время) |
| **Дерево неисправностей** | Проблема → Причина → Решение |

### Пример записи в базе неисправностей

```javascript
{
    problem: 'Отслоение (warpping)',
    causes: [
        'Недостаточная адгезия первого слоя',
        'Слишком высокая скорость охлаждения',
        'Неправильная температура стола'
    ],
    solutions: [
        'Откалибруйте стол (особенно первый слой)',
        'Используйте клей/лак для адгезии',
        'Увеличьте температуру стола на 5-10°C'
    ],
    safetyWarning: 'При очистке стола используйте перчатки!'
}
```

## 🔌 ИНТЕГРАЦИЯ

### Подключение модуля

```html
<!-- В index.html -->
<script src="src/modules/ai-engineer.js"></script>
```

### Инициализация

```javascript
// Создание экземпляра
const ai = new AIEngineer();

// Настройка callbacks
ai.onStatusChange = (status) => {
    console.log('Статус:', status.message);
};

ai.onMessageReceived = (message) => {
    displayMessage(message.response, message.type);
};

// Инициализация
await ai.initialize();
```

### Обновление контекста

```javascript
// Синхронизация с приложением
ai.updateContext({
    currentTab: 'calculator',
    selectedPrinter: printerData,
    selectedMaterial: materialData,
    currentOrder: orderData
});

// Передача данных парка принтеров
ai.setPrinters(printerSystem.getAllPrinters());

// Передача данных склада материалов
ai.setMaterials(materialsModule.getAllMaterials());
```

## 💻 ИСПОЛЬЗОВАНИЕ

### Отправка сообщения

```javascript
// Пользователь вводит сообщение
const message = 'Как настроить температуру для ABS?';

// Отправка
const result = await ai.sendMessage(message);

// Обработка ответа
if (result.success) {
    if (result.type === 'safety_warning') {
        showWarning(result.response);
    } else {
        showResponse(result.response);
    }
}
```

### Классификация запросов

ИИ автоматически классифицирует запросы:

| Категория | Ключевые слова | Пример |
|-----------|---------------|--------|
| `printer_info` | принтер, устройство, park | «Какой принтер выбрать?» |
| `material_info` | материал, пластик, смола | «Температура PLA» |
| `troubleshooting` | проблема, отслой, засор | «Отслаивается первый слой» |
| `safety` | безопасн, токсичн, перчатк | «Безопасность с ABS» |
| `calculation` | температур, скорост, слой | «Параметры для PETG» |
| `greeting` | привет, здравствуй | «Привет!» |
| `unknown` | — | «Расскажи анекдот» |

## ⚙️ НАСТРОЙКА LLM-МОДЕЛИ

### Рекомендуемые модели

| Модель | Размер | Качество | Скорость |
|--------|--------|----------|----------|
| **TinyLlama-1.1B** | ~800 MB | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **Qwen-1.8B** | ~1.2 GB | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| **Phi-2-2.7B** | ~1.8 GB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

### Загрузка модели (Electron)

1. Откройте настройки ИИ (⚙️ в окне чата)
2. Нажмите «📥 Загрузить LLM модель (GGUF)»
3. Выберите файл модели (.gguf)
4. Дождитесь завершения загрузки

### Требования к модели

- **Формат:** GGUF (квантованный)
- **Размер:** 0.5 — 4.0 ГБ
- **Квантование:** Q4_K_M или выше (рекомендуется)

### Интеграция с llama.cpp

Для полной интеграции локальной модели:

```javascript
// main.js (Electron)
const { spawn } = require('child_process');

ipcMain.handle('generate-ai-response', async (event, prompt) => {
    const modelPath = getSavedModelPath();
    
    return new Promise((resolve) => {
        const llama = spawn('./llama-cli', [
            '-m', modelPath,
            '-p', prompt,
            '-n', '512',
            '--temp', '0.3'
        ]);
        
        let response = '';
        llama.stdout.on('data', (data) => {
            response += data.toString();
        });
        
        llama.on('close', () => {
            resolve({ success: true, response });
        });
    });
});
```

## 📖 API REFERENCE

### AIEngineer

#### Методы

| Метод | Описание | Возвращает |
|-------|----------|------------|
| `initialize()` | Инициализация системы | `Promise<Object>` |
| `sendMessage(message)` | Отправка сообщения | `Promise<Result>` |
| `updateContext(context)` | Обновление контекста | `void` |
| `setPrinters(printers)` | Установка парка принтеров | `void` |
| `setMaterials(materials)` | Установка материалов | `void` |

#### Свойства

| Свойство | Тип | Описание |
|----------|-----|----------|
| `isLoaded` | boolean | Готовность системы |
| `isLoading` | boolean | Процесс загрузки |
| `modelPath` | string | Путь к модели |
| `chatHistory` | Array | История чата |
| `safetyLayer` | SafetyLayer | Система безопасности |
| `knowledgeBase` | KnowledgeBase | База знаний |

### SafetyLayer

#### Методы

| Метод | Описание | Возвращает |
|-------|----------|------------|
| `validateUserInput(input)` | Проверка запроса | `{safe: boolean, warning?: string}` |
| `validateResponse(response)` | Валидация ответа | `{safe: boolean, warning?: string}` |

### KnowledgeBase

#### Методы

| Метод | Описание | Возвращает |
|-------|----------|------------|
| `initialize()` | Загрузка данных | `Promise<void>` |
| `search(query, intent)` | Поиск данных | `KnowledgeResult` |
| `searchPrinters(name)` | Поиск принтеров | `Printer[]` |
| `searchMaterials(name)` | Поиск материалов | `Material[]` |
| `searchTroubleshooting(problem)` | Поиск неисправностей | `Troubleshooting[]` |

## 🔧 ТЕСТИРОВАНИЕ

### Проверка системы безопасности

```javascript
// Тест опасного запроса
const dangerousQuery = 'Как починить сопло пока оно горячее?';
const result = await ai.sendMessage(dangerousQuery);

// Ожидаемый результат: safety_warning
console.assert(result.type === 'safety_warning');
console.assert(result.blocked === true);
```

### Проверка базы знаний

```javascript
// Тест запроса о материале
const materialQuery = 'Температура печати ABS';
const result = await ai.sendMessage(materialQuery);

// Проверка наличия данных
console.assert(result.source === 'knowledge_base');
console.assert(result.response.includes('°C'));
```

## 📝 ЗАМЕТКИ

### Производительность

- **Режим базы знаний:** Мгновенные ответы (<100ms)
- **Режим LLM:** Зависит от модели (1-10 секунд)
- **Потребление памяти:** ~100 MB (без LLM), ~2 GB (с LLM)

### Ограничения

1. ИИ не заменяет официальную документацию
2. При отсутствии данных в базе — рекомендует обратиться к специалисту
3. Не дает советов вне сферы 3D-печати
