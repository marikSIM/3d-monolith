# 🏗️ AI SUPPORTS — Анализ поддержек для 3D-печати

## Описание
Модуль **AI SUPPORTS** автоматически анализирует 3D-модель и определяет зоны, требующие поддержек (support structures) для успешной печати.

## Возможности

### 🔍 Анализ модели
- **Определение свесов** (overhangs) с углом >45°
- **Классификация по серьёзности**:
  - 🔴 **Критические** (>60°) — требуют обязательных поддержек
  - 🟠 **Проблемные** (50-60°) — желательны поддержки
  - 🟡 **Незначительные** (45-50°) — можно печатать без поддержек

### 🎨 Визуализация
- **Красным цветом** — критические зоны
- **Оранжевым цветом** — проблемные зоны
- **Жёлтым цветом** — незначительные свесы

### 💡 Рекомендации
Автоматическая генерация рекомендаций:
- Тип поддержек (Tree/Normal)
- Плотность поддержек (%)
- Z-расстояние (мм)
- Interface layers (количество слоёв)
- Ориентация модели

## Использование

### Через чат с ИИ
```
👤 поддержки
👤 свесы
👤 где нужны поддержки
👤 анализ поддержек
👤 overhang analysis
```

### Через кнопку
После загрузки модели в правом нижнем углу 3D-вьюера появляется кнопка:
- **🏗️ Показать поддержки** — включить визуализацию
- **🔒 Скрыть поддержки** — выключить визуализацию

### Программно
```javascript
// Анализ модели
window.AI_SUPPORTS.analyzeSupports(model, layerHeight);

// Визуализация
window.AI_SUPPORTS.visualizeOverhangs(model, scene);

// Скрыть визуализацию
window.AI_SUPPORTS.hideVisualization(scene);

// Получить отчёт
const report = window.AI_SUPPORTS.getReport();
console.log(report.report);
```

## Интеграция с AI Vision

Модуль автоматически запускается после анализа модели через **AI Vision**:
1. Пользователь загружает STL файл
2. AI Vision анализирует размеры, объём, сложность
3. AI SUPPORTS автоматически определяет свесы
4. Пользователь может запросить анализ через чат

```
• Проблемные зоны: 89
• Незначительные: 100

💡 Рекомендации:

   Много свесов. Увеличьте плотность для надёжности.

🔄 Ориентация модели
   Рассмотрите поворот на 45° для уменьшения свесов.
```

## Настройки по умолчанию

| Параметр | Значение |
|----------|----------|
| Угол свеса | >45° |
| Критический угол | >60° |
| Высота слоя | 0.2 мм |
| Z-расстояние | 0.8 × слой |
| Плотность | 10-15% |
