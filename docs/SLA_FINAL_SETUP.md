## 🔧 Вариант 1: Установка .NET 8 (для UVTools)

**Зачем:** UVTools 6.x даст **100% точность** всех параметров.

### Установка:

1. **Запустите скрипт:**
   ```bash
   install-dotnet8.bat
   ```

2. **ИЛИ скачайте вручную:**
   - Перейдите на https://dotnet.microsoft.com/download/dotnet/8.0
   - Скачайте **ASP.NET Core Runtime 8.0** (x64)
   - Установите

3. **Проверьте:**
   ```cmd
   dotnet --list-runtimes
   ```
   Должно показать: `Microsoft.AspNetCore.App 8.0.x`

4. **Перезапустите приложение** и попробуйте загрузить PWMO снова.

## 📝 Формулы расчёта:

### Время печати:
```
Время = (Днищные_слои × 60) + (Обычные_слои × (Экспозиция + 21.5))
```

### Объём смолы:
```
Объём = (Ширина × Высота × Толщина) × 0.21 × 1.1
```

### Вес смолы:
```
Вес = Объём × 1.1 (плотность смолы)
```
