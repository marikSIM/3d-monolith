## ⚡ БЫСТРЫЙ СТАРТ

### Вариант 1: Перед закрытием работы (рекомендуется)

**Запустите один из скриптов:**

```powershell
cd "C:\Users\murad\Desktop\3D MONOLITH"

# Быстрое сохранение
.\quick-push.bat

# Или авто-бэкап с датой
.\auto-backup.bat
```

**Занимает:** 10-30 секунд

---

### Вариант 2: Автоматически при закрытии

**Настройте Task Scheduler (Планировщик заданий):**

1. Откройте **Task Scheduler** (Пуск → "Планировщик заданий")

2. **Create Basic Task** (Создать простую задачу)

3. **Name:** `3D MONOLITH Auto Backup`

4. **Trigger:** 
   - Выберите **Daily** (Ежедневно)
   - Время: `18:00` (или когда заканчиваете работу)

5. **Action:**
   - **Start a program**
   - Program: `C:\Users\murad\Desktop\3D MONOLITH\auto-backup.bat`
   - Start in: `C:\Users\murad\Desktop\3D MONOLITH`

6. **Finish**

**Теперь будет сохраняться автоматически каждый день!**

---

### Вариант 3: При закрытии VS Code

**Установите расширение:**

1. Откройте VS Code
2. Extensions (Ctrl+Shift+X)
3. Найдите: **Git Auto Commit**
4. Установите

**Настройте `.vscode/settings.json`:**

```json
{
    "git.autoCommit": true,
    "git.autoCommitInterval": 300000,
    "git.autoCommitMessage": "Auto-save: "
}
```

## 📝 ПАМЯТКА

### Что сохраняется автоматически:
- ✅ Изменения в коде (`.html`, `.js`, `.css`)
- ✅ Документация (`.md`)
- ✅ Конфигурация (`.json`)

### Что НЕ сохраняется:
- ❌ `node_modules/` (восстанавливается через `npm install`)
- ❌ `*.log` (логи)
- ❌ `*.zip` (большие файлы)
- ❌ Временные файлы

## ⚠️ ВАЖНО!

**Git — это не облачное хранилище!**

| Действие | Google Drive/OneDrive | Git/GitHub |
|----------|----------------------|------------|
| **Автосохранение** | ✅ Да | ❌ Нет |
| **Резервная копия** | ✅ Да | ❌ Только после `push` |
| **История версий** | ✅ Частично | ✅ Полная |
| **Откат изменений** | ⚠️ Ограниченно | ✅ Полный |

**Вывод:** Сохраняйте вручную перед закрытием!

## 🆘 ЕСЛИ ЗАБЫЛИ СОХРАНИТЬ

### Локальные изменения есть:
```powershell
git status          # Показать изменения
git add .           # Добавить всё
git commit -m "..." # Сохранить
git push            # Отправить
```

### Если компьютер сломался:
- ❌ Без `git push` изменения потеряны
- ✅ С `git push` всё на GitHub
