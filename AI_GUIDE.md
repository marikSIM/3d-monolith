# 🚀 РУКОВОДСТВО ПО ГИБРИДНОЙ AI СИСТЕМЕ

## ✅ Что было реализовано

### Три режима работы AI:

1. **📚 База знаний (Knowledge Base)**
   - Работает сразу после запуска
   - Мгновенные ответы из проверенной базы
   - Не требует дополнительного ПО
   - **Режим по умолчанию**

2. **🤖 Локальная LLM (node-llama-cpp)**
   - Полноценная языковая модель
   - Работает локально на вашем устройстве
   - Требует GGUF модель (2-8 ГБ)
   - Умные ответы с пониманием контекста

3. **🌐 WebLLM**
   - Браузерная модель через WebGPU
   - Не требует загрузки модели
   - Требуется поддержка WebGPU
   - Современная архитектура

## 🎯 Как использовать

### 1. Запуск приложения

```bash
npm start
```

### 2. Открытие AI чата

1. Нажмите на кнопку **🔧 Fixik AI** в правом нижнем углу
2. Откроется окно чата

### 3. Переключение режимов

Внизу окна чата находится переключатель:

```
┌─────────────────────────────┐
│  📚    🤖    🌐            │
│  База  LLM   Web           │
├─────────────────────────────┤
│ ✓ Режим готов               │
│ База знаний — быстро...     │
└─────────────────────────────┘
```

**Кнопки:**
- **📚 База** — база знаний (всегда доступна)
- **🤖 LLM** — локальная модель (требует загрузку)
- **🌐 Web** — WebLLM (требует WebGPU)

### Шаг 1: Скачайте GGUF модель

Рекомендуемые модели:

| Модель | Размер | Качество | Ссылка |
|--------|--------|----------|--------|
| **TinyLlama-1.1B** | ~2 ГБ | Базовое | [Скачать](https://huggingface.co/TheBloke/TinyLlama-1.1B-Chat-v1.0-GGUF/resolve/main/tinyllama-1.1b-chat-v1.0.Q4_K_M.gguf) |

| **Llama-2-7B** | ~4 ГБ | Хорошее | [Скачать](https://huggingface.co/TheBloke/Llama-2-7B-Chat-GGUF/resolve/main/llama-2-7b-chat.Q4_K_M.gguf) |

### Шаг 2: Загрузите в приложение

1. Откройте AI чат
2. Нажмите **🤖 LLM**
3. Появится диалог выбора файла
4. Выберите `.gguf` файл
5. Дождитесь загрузки (5-30 секунд)

### Шаг 3: Используйте

- Модель автоматически активируется
- Путь сохраняется в настройках
- Автоматическая загрузка при следующем запуске

## ⚠️ Возможные проблемы

### Кнопка LLM серая (недоступна)

**Причина:** Модель не загружена

**Решение:**
1. Нажмите на кнопку LLM
2. Загрузите GGUF модель
3. Или переключитесь на базу знаний

### Кнопка Web серая (недоступна)

**Причина:** WebGPU не поддерживается

**Решение:**
1. Проверьте поддержку WebGPU:
   ```javascript
   console.log(navigator.gpu)
   ```
2. Обновите драйверы GPU
3. Используйте базу знаний или LLM

### Модель долго отвечает

**Причина:** Слабое железо или большая модель

**Решение:**
1. Используйте меньшую модель (1-2 ГБ)
2. Уменьшите `maxTokens` в настройках
3. Переключитесь на базу знаний

### Ошибка при загрузке модели

**Причина:** Неподдерживаемый формат

**Решение:**
1. Убедитесь что файл `.gguf`
2. Проверьте размер (0.5-8 ГБ)
3. Скачайте модель заново
