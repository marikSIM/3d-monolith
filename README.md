## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/marikSIM/3d-monolith.git
cd 3d-monolith

# Установите зависимости
npm install

# Запустите приложение
npm start
```

### 2. Требования

- **Node.js** 16+ — https://nodejs.org/
- **Electron** — устанавливается автоматически
- **Windows 10/11** (тестировалось на Windows 11)

## 🔧 Дополнительные компоненты

### Для SLA печати (фотополимер):

**UVTools** — для точного расчёта времени и материалов:

1. Скачайте: https://github.com/sn4k3/UVtools/releases
2. Распакуйте в `C:\Program Files\UVtools\`
3. Добавьте в PATH или укажите путь в настройках приложения

### Для работы AI-ассистента:

**Локальная LLM** (опционально):

1. Скачайте GGUF модель (TinyLlama, Mistral и др.)
2. Загрузите через интерфейс AI-чата
3. Или используйте базу знаний (работает без модели)

## 🎮 Использование

### Калькулятор FDM

1. Выберите технологию **FDM**
2. Загрузите STL модель
3. Выберите материал (PLA, PETG, ABS и др.)
4. Настройте параметры (слой, заполнение)
5. Нажмите **Рассчитать**

### Калькулятор SLA

1. Выберите технологию **SLA**
2. Загрузите файл (.pwmo, .ctb, .photon, .sl1)
3. Система автоматически извлечёт:
   - Объём смолы
   - Время печати
   - Количество слоёв
4. Рассчитайте стоимость

### CRM

1. Вкладка **Клиенты** → Добавить клиента
2. Вкладка **Заказы** → Создать заказ
3. Привяжите модель к заказу
4. Отслеживайте статус

## 🤖 AI-ассистент

Нажмите на кнопку **🔧 Fixik AI** в правом нижнем углу.

**Режимы работы:**
- 📚 **База знаний** — мгновенные ответы (всегда доступно)
- 🤖 **Локальная LLM** — развёрнутые ответы (требует модель)
- 🌐 **WebLLM** — браузерная модель (требует WebGPU)
