## 📥 2 СПОСОБА ЗАГРУЗКИ МОДЕЛИ

### Способ 1: Быстрая загрузка (если модель уже скачана)

1. **Откройте AI чат** (нажмите 🔧 внизу экрана)

2. **Нажмите ⚙️** (кнопка настроек в окне чата)

3. **Выберите "🤖 Загрузить LLM модель"**

4. **Выберите .gguf файл** в диалоге

5. **Готово!** Модель автоматически активируется

---

### Способ 2: Скачать модель через браузер

#### Шаг 1: Откройте AI чат
Нажмите 🔧 внизу экрана

#### Шаг 2: Переключитесь на LLM
Нажмите кнопку **🤖 LLM** внизу чата

#### Шаг 3: Получите инструкцию
В чате появится сообщение с инструкцией и ссылками

#### Шаг 4: Скачайте модель
Нажмите на ссылку модели:
- **TinyLlama-1.1B** (~2 ГБ) — базовый уровень
- **Mistral-7B** (~4 ГБ) — хорошее качество

#### Шаг 5: Откройте ссылку
В чате появится кнопка **"📥 Открыть ссылку для скачивания"**

#### Шаг 6: Дождитесь загрузки
Файл сохранится в папке **"Загрузки"**

#### Шаг 7: Загрузите модель в приложение
1. Нажмите ⚙️ в окне чата
2. Выберите "🤖 Загрузить LLM модель"
3. Выберите скачанный `.gguf` файл
4. Дождитесь загрузки

## 🐛 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### Ошибка: "Файл не найден"

**Причина:** Файл не существует или перемещён

**Решение:**
1. Убедитесь что файл `.gguf`
2. Не перемещайте файл после загрузки
3. Загрузите модель заново

### Ошибка: "Размер модели вне диапазона"

**Причина:** Файл повреждён или не GGUF

**Решение:**
1. Скачайте модель заново
2. Проверьте размер (0.5-8 ГБ)
3. Убедитесь что это GGUF формат

### Модель долго отвечает

**Причина:** Слабое железо или большая модель

**Решение:**
1. Используйте TinyLlama (2 ГБ)
2. Закройте другие программы
3. Переключитесь на базу знаний 📚

### Кнопка LLM серая

**Причина:** Модель не загружена

**Решение:**
1. Нажмите ⚙️ → Загрузить модель
2. Выберите .gguf файл
3. Дождитесь загрузки
