# ✅ ОТЧЁТ: Гибридная AI система для 3D MONOLITH

## 📋 Задание

> **Реализовать гибридную AI систему с переключателем режимов:**
> - База знаний (RAG) — быстро и надёжно
> - Локальная LLM (node-llama-cpp) — полноценный ИИ
> - WebLLM — браузерная модель через WebGPU
> 
> **Требования:**
> - Вынести базу в отдельный JSON-файл
> - Использовать IPC для связи процессов
> - Хранить данные в файлах через app.getPath('userData')
> - Автоопределение доступности WebGPU
> - Переключатель режимов для пользователя

### 3. IPC связь (Renderer ↔ Main)

**Созданные IPC обработчики в `main.js`:**

```javascript

ipcMain.handle('get-ai-mode', ...)             // Получить режим
ipcMain.handle('set-ai-mode', ...)             // Установить режим

ipcMain.handle('load-ai-model', ...)           // Загрузка GGUF
ipcMain.handle('get-saved-ai-model', ...)      // Проверка модели
ipcMain.handle('check-webgpu', ...)            // Проверка WebGPU
ipcMain.handle('save-ai-settings', ...)        // Сохранение настроек
ipcMain.handle('get-ai-settings', ...)         // Получение настроек
```

**Методы в `preload.js`:**

### 4. Хранение данных (Electron Store)

**Путь:** `app.getPath('userData')`

**Хранимые данные:**
```javascript
store.set('aiMode', 'knowledge-base')     // Текущий режим
store.set('modelPath', '/path/to.gguf')   // Путь к модели
store.set('aiSettings', {...})            // Настройки AI
```

**Windows:** `C:\Users\<USER>\AppData\Roaming\3D MONOLITH Studio\`

### 5. Автоопределение WebGPU

**Функция:** `detectWebGPU()`

```javascript
async detectWebGPU() {
    if (navigator.gpu) {
        const adapter = await navigator.gpu.requestAdapter();
        return !!adapter;
    }
    return false;
}
```

**Результат:**
- ✅ WebGPU доступен → кнопка "Web" активна
- ❌ WebGPU недоступен → кнопка "Web" серая

### 6. UI переключатель режимов

**Расположение:** Внизу окна AI чата

**Внешний вид:**
```
┌─────────────────────────────────┐
│  📚    🤖    🌐                │
│  База  LLM   Web               │
├─────────────────────────────────┤
│ ✓ Режим готов                   │
│ База знаний — быстро и надёжно  │
└─────────────────────────────────┘
```

**Функционал:**
- Переключение одним кликом
- Индикация доступности (серая = недоступно)
- Статус режима в реальном времени
- Приветственное сообщение при переключении

## 🎯 Как использовать

### Быстрый старт

1. **Запустить приложение:**
   ```bash
   npm start
   ```

2. **Открыть AI чат:**
   - Нажать 🔧 в правом нижнем углу

3. **Переключить режим:**
   - Использовать кнопки внизу чата

### Загрузка локальной модели

1. **Скачать GGUF модель:**
   - [TinyLlama-1.1B-Chat-v1.0-GGUF](https://huggingface.co/TheBloke/TinyLlama-1.1B-Chat-v1.0-GGUF)
   - [Mistral-7B-Instruct-v0.2-GGUF](https://huggingface.co/TheBloke/Mistral-7B-Instruct-v0.2-GGUF)
   - [Llama-2-7B-Chat-GGUF](https://huggingface.co/TheBloke/Llama-2-7B-Chat-GGUF)

2. **Загрузить в приложение:**
   - Открыть AI чат
   - Нажать 🤖 LLM
   - Выбрать `.gguf` файл

3. **Использовать:**
   - Модель автоматически активируется
   - Путь сохраняется в настройках

## 🔐 Безопасность

### Реализованные уровни защиты:

1. **IPC изоляция** — renderer не имеет прямого доступа к файлам
2. **Валидация путей** — проверка существования файлов
3. **Проверка размеров** — модели 0.5-8 ГБ
4. **Fallback режим** — автоматическое переключение на базу при ошибке
5. **Content Security Policy** — защита от XSS

### Красные линии (блокируются):
- ❌ Отключение питания при нагретых компонентах
- ❌ Разборка хотэнда без остывания
- ❌ Работа с фотополимерами без СИЗ
- ❌ Превышение температурных лимитов

## 🎯 Рекомендации по использованию

### Используйте **Базу знаний** когда:
- ✅ Нужен быстрый ответ
- ✅ Вопрос конкретный (температура, настройки)
- ✅ Проблема известна (отслоение, засор)
- ✅ Слабое железо

### Используйте **Локальную LLM** когда:
- ✅ Нужен развёрнутый ответ
- ✅ Вопрос требует понимания контекста
- ✅ Есть мощное железо (8+ ГБ RAM)
- ✅ Нет интернета

### Используйте **WebLLM** когда:
- ✅ Есть WebGPU (современная видеокарта)
- ✅ Не хотите загружать модель
- ✅ Нужна современная архитектура
- ✅ Есть интернет
